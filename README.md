## 🚀 How to Use This Repository

### Clone the Repository
```bash
git clone <repository-url>
cd java
```

### Compile and Run
```bash
# Navigate to a topic folder
cd 1_variable

# Compile
javac javabasic.java

# Run
java javabasic
```
